# Smart Blood Bank Management System

Smart Blood Bank Management System is a role-based web application built with Spring Boot, Spring Security, JPA, Thymeleaf, and MySQL. It helps connect donors, hospitals, and admins in one platform for managing blood requests, donor matching, notifications, and inventory monitoring.

## Key Features
- Role-based login for Admin, Donor, and Hospital
- Donor registration and profile management
- Hospital blood request creation and tracking
- Blood group matching for compatible donors
- Automatic donor notifications for urgent requests
- Donor accept/decline response handling
- Admin dashboard with reports and analytics
- Responsive UI built with Bootstrap and Thymeleaf

## Design Patterns Used
- Strategy Pattern for blood matching
- Observer Pattern for donor notifications
- Factory Pattern for user creation concept
- MVC, Repository, and Service layer architecture

## Tech Stack
- Backend: Spring Boot, Spring Security, Spring Data JPA
- Frontend: Thymeleaf, Bootstrap, HTML, CSS, JavaScript
- Database: MySQL

## Project Goal
The main goal of this project is to make blood request coordination faster, more organized, and easier to manage for hospitals, donors, and administrators.
